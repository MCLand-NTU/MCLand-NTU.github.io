A Python software tool to plot and visualize Waddington's Epigenetic Landscape

Version 1.0.0 requires Python 3

MCLand is a graphical user interface software tool that allows user to plot and visualize Waddington's epigenetic landscape for gene regulatory network using Monte Carlo method

The MCLand software tool is for research and academic purposes only. We make no warranty or guarantee of the use of the software.
